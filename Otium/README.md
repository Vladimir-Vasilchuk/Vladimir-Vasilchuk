# Otium: внедрение сортировки через API

## Описание проекта

Доработка API стримингового сервиса Otium: реализация функции сортировки контента. Проект демонстрирует навыки работы с требованиями заказчика, проектирование REST API, UML‑моделирование (диаграмма последовательности), тестирование API (Swagger), командное взаимодействие.

Цель проекта:
* Изучить требования заказчика на основе интервью
* Создать диаграмму последовательности (UML) для визуализации взаимодействия компонентов при сортировке контента
* Протестировать существующее API в Swagger и оценить его готовность к доработке
* Сформулировать задачу на разработку новой операции API с чёткими критериями приёмки

Этапы проектирования:
* Анализ требований
* Создание диаграммы последовательности (UML)  для описания сценария сортировки
* Тестирование существующего API (проверка текущих эндпоинтов в Swagger)
* Постановка задачи на разработку API
