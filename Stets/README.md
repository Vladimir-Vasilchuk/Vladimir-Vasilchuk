# Stets - дипломный проект 

## Описание проекта

Проект Stets — производитель устройств умного дома. Основной задачей проекта стало проектирование и разработка мобильного приложения Stets Home, обеспечивающего расширенные возможности управления умными устройствами с уникальной функцией энергосбережения.

Этапы проектирования:
* Разработка требований к приложению
* Разработка и демонстрация прототипа пользовательского интерфейса приложения
* Разработка программы и методики испытаний и словаря данных
* Видеопрезентация приложения
