# Разработка ПМИ и руководства пользователя для сервиса «Капсула»

## Описание проекта

В рамках проекта разработаны ключевые артефакты для верификации исправлений на сайте бронирования капсульных отелей «Капсула»: программа и методика испытаний (ПМИ) и руководство пользователя.

Цель проекта:
* Разработать ПМИ для тестирования исправленных ошибок на сайте бронирования
* Создать руководство пользователя по ГОСТ 19, описывающее все функции сервиса.

Этапы проектирования:
* Анализ ранее составленного плана тестирования
* Составление ПМИ: определение предусловий, описание шагов проверки по тест‑кейсам, формулирование конечных результатов для требований
* Написание руководства пользователя: описание всех функций сайта, структурирование материала с оглавлением и оформление по ГОСТ 19 с использованием понятных формулировок для конечного пользователя.
